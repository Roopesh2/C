(static) HSLToRGB(hue, saturation, lightness) → {Array.<number>}

Converts an HSL color value to RGB. Conversion formula adapted from http://en.wikipedia.org/wiki/HSL_color_space. Assumes hue is between 0 and 360, saturation and lightness are between 0 & 1 and returns red, green, and blue values in the range 0 to 1
